Main Projects

Guide To Italy

I recently created a Guide to Italy website using HTML, CSS, and JavaScript, focusing on its food, culture, and history. With HTML and CSS, I designed a clean layout highlighting Italy's culinary delights, cultural festivals, and iconic landmarks. JavaScript added interactive features like image galleries and timelines, enhancing the user experience. It was a rewarding project that allowed me to showcase Italy's diverse offerings and create a platform for others to explore its rich heritage.

GuideMe

I led the creation of a WordPress website for GuideMe, a support group for the blind. Using WordPress, I ensured the site was easy to navigate and accessible to all. It offers essential resources and a supportive community for visually impaired individuals. I'm proud to have helped GuideMe in their mission to empower and assist the blind community online.
